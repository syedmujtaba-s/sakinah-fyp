Sakinah v1.0.3 — habit title fix.

Fixed:
- Habit titles created from "Track this advice" were truncated to 42
  characters with a literal "..." stored in the database, so the habit
  detail screen showed something like "Reflect on how the Prophet's
  smile made ot..." Now stores the full advice text and wraps it
  cleanly across multiple lines on the detail screen.

Behavior for old habits:
- Existing habits with the truncated title automatically fall back to
  showing the full text (sourceAdvice field). No data migration needed.

Carried over from v1.0.2:
- Off-topic detector (factual questions, code etc. now show friendly
  redirect instead of forcing a Seerah connection)
- Delete journal entries with optional bookmarks/habits cascade

If anything looks off, hit Send Feedback and it'll reach the admin panel.
